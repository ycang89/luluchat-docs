# Tickets

## What is Tickets?
The Tickets module helps you organize and manage your customer support and internal processes. You can create multiple pipelines to track different types of requests, ensuring every task is tracked from start to finish.

## Core Features
- **Pipelines & Stages**: Create custom workflows that match your business processes.
- **Kanban Board**: A visual way to track the progress of tickets by dragging them across stages.
- **Reporting**: Detailed summaries of your team's workload, priority distribution, and performance.
- **Automation**: Move tickets through stages automatically via Message Flows.

## Module Structure
- [Summary](./summary.md): High-level reports and overview of your ticketing pipelines.
- [Board](./board.md): The visual Kanban interface for managing active tickets.
- [List](./list.md): A searchable table view of all tickets in a pipeline.
