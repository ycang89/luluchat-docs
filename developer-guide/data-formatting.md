# Data Formatting

For all the parameters that used to communicate or send between Luluchat.io webhook / API and other third party platform must fulfil following format:

<table data-full-width="false"><thead><tr><th width="144">Value Type</th><th width="604.3333333333333">Description</th></tr></thead><tbody><tr><td>String</td><td>Any string value where support up to maximum length of 255 characters</td></tr><tr><td>Numeric</td><td>Any numeric value (integer, float)<br><br>Example: 1, 12.1234</td></tr><tr><td>Date</td><td>The date format we used are following <strong>Year-Month-Date</strong> format, where the month and date value should precede with 0 (Zero) if it's a single value.<br><br><strong>Format: YYYY-MM-DD</strong><br>Example: 2023-02-01 (Represent 1st of February, Year 2023)</td></tr></tbody></table>
