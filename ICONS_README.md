# Menu Icons

This directory contains SVG icons for the GitBook menu navigation. All icons use a consistent color scheme with the primary color `#6366F1` (Indigo).

## Icon Files

- `introduction.svg` - For Introduction section
- `getting-started.svg` - For Getting Started section
- `dashboard.svg` - For Dashboard
- `inbox.svg` - For Inbox
- `automations.svg` - For Automations
- `broadcasts.svg` - For Broadcasts
- `contacts.svg` - For Contacts
- `message-templates.svg` - For Message Templates
- `reports.svg` - For Reports
- `forms.svg` - For Forms
- `bookings.svg` - For Bookings
- `tickets.svg` - For Tickets
- `deals.svg` - For Deals
- `settings.svg` - For Settings

## Usage in GitBook

For GitBook.com, you can:
1. Upload these icons through the GitBook UI in Space Settings > Appearance > Custom Icons
2. Or use the CSS file in `styles/website.css` to automatically apply icons to menu items

All icons are 24x24px and use the same indigo color (#6366F1) for visual consistency.
