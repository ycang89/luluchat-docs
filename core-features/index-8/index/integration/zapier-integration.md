# Zapier Integration

To start using luluchat in zapier, you have to accept following invitation. Please click the link below and start building your own zap.\
luluchat: [https://zapier.com/developer/public-invite/186610/ed983254887d7d19937a08a198b5f5f9/](https://zapier.com/developer/public-invite/186610/ed983254887d7d19937a08a198b5f5f9/)

<figure><img src="../../../../.gitbook/assets/Screenshot 2024-07-12 at 2.14.22 PM.png" alt=""><figcaption></figcaption></figure>

**Step 1: Choose your Trigger, in here we use Calendly as an example. To choose luluchat as the Action, click on the plus icon to Add a Step.**

<figure><img src="../../../../.gitbook/assets/Screenshot 2024-07-12 at 1.01.01 PM.png" alt=""><figcaption></figcaption></figure>



**Step 2: Search for luluchat**

<figure><img src="../../../../.gitbook/assets/Screenshot 2024-07-12 at 1.01.16 PM.png" alt=""><figcaption></figcaption></figure>



**Step 3: Choose Send Message Flow in the Event Selection**

<figure><img src="../../../../.gitbook/assets/Screenshot 2024-07-12 at 1.01.29 PM.png" alt=""><figcaption></figcaption></figure>



**Step 4: Connect your account to luluchat platform**

There will a pop up and ask for you Zapier API Key. Look for your Zapier API Key from your account in luluchat platform.

<figure><img src="../../../../.gitbook/assets/Screenshot 2024-07-12 at 1.02.05 PM.png" alt=""><figcaption></figcaption></figure>

Go to [Settings > Integration](https://app.luluchat.io/settings?view=integration), Copy the Zapier API Key and paste to the pop up and proceed.

<figure><img src="../../../../.gitbook/assets/Screenshot 2024-07-12 at 1.02.26 PM.png" alt=""><figcaption></figcaption></figure>

<figure><img src="../../../../.gitbook/assets/Screenshot 2024-07-12 at 1.02.48 PM.png" alt=""><figcaption></figcaption></figure>



**Step 5: Select your Message Flow that you have created in luluchat Platform. If you haven't do so, please create one in** [**Automation > Message Flows**](https://app.luluchat.io/automations/flows)**.**

<figure><img src="../../../../.gitbook/assets/Screenshot 2024-07-12 at 1.03.10 PM.png" alt=""><figcaption></figcaption></figure>

**In your message flow, add "Webhook" as trigger in the Starting Step, this will open up the connection for Zapier to call.**

<figure><img src="../../../../.gitbook/assets/Screenshot 2024-07-19 at 3.34.14â€¯PM.png" alt=""><figcaption></figcaption></figure>



**Step 6: Define your Action, in this step you will be configure which message flow to send out and send to which phone number.**

Select the message flow you have created at luluchat Platform

<figure><img src="../../../../.gitbook/assets/Screenshot 2024-07-12 at 1.04.35 PM.png" alt=""><figcaption></figcaption></figure>

Select the phone number from data provided by the Trigger ( in this case Phone Number provided by Calendly )

<figure><img src="../../../../.gitbook/assets/Screenshot 2024-07-12 at 1.04.58 PM.png" alt=""><figcaption></figcaption></figure>



**Step 7: You may test your integration in this step, or skip test and publish it for production use.**
