# ChatGPT Integration

To integrate the luluchat app with **ChatGPT**, you **must have a ChatGPT account**. Additionally, you will be required to **provide the Secret Key** for the luluchat app integration with ChatGPT.

## Sign Up a ChatGPT account

**Step 1**: Sign up or Log in **ChatGPT** account in [**https://platform.openai.com**](https://platform.openai.com)**.**

**Step 2**: Go to [**https://platform.openai.com/account/api-keys**](https://platform.openai.com/account/api-keys)**,** Click **Create new secret key** to get API Keys.
