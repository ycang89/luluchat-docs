# Forms: Building a Form

## How to Build a Form (Step 1)
Building a form in Luluchat is a simple drag-and-drop experience. You can add various elements to collect exactly the data you need.

### Adding Elements
Click the **Build Form** tab to start adding components:
- **Title**: Add a heading and description for your form.
- **Text/TextArea**: For names, open-ended feedback, or short notes.
- **Multiple Choice (Radio/Checkbox/Dropdown)**: For specific options or categorizing leads.
- **Phone Number**: Collect additional contact numbers (automatically guesses country codes).
- **Date & Time**: For booking requests or availability.
- **Media (Image/Video)**: Add visual instructions or promotional content to your form.
- **File Upload**: Allow customers to upload receipts, documents, or photos.

{% hint style="warning" %}
**File Storage Required**: To use the File Upload element in your forms, you need to purchase the "File Storage 200G" add-on from your plan page. Go to `Settings` > `Account Management` > `Billing` to view and purchase storage add-ons.
{% endhint %}

### Mapping to Custom Attributes
For every question, you can link the answer to a **Custom Attribute**. This means as soon as the customer submits the form, their contact profile in Luluchat is updated automatically.

## Important behavior to know

* **File Upload Storage**: The File Upload element requires the "File Storage 200G" add-on to be purchased. Without this add-on, you cannot use file upload fields in your forms. Go to `Settings` > `Account Management` > `Billing` to purchase the storage add-on.
* **File Size Limits**: Uploaded files are subject to size limits based on your storage plan. Check your plan details for specific limits.
* **Supported File Types**: File uploads support common document, image, and video formats. Check the form builder for the complete list of supported file types.

- **Keep it Short**: Only ask for essential information to increase submission rates.
- **Use Clear Labels**: Make sure your questions are easy to understand.
- **Required Fields**: Mark critical fields (like Name or Email) as "Required" to ensure you get the data you need.
- **Plan Ahead for File Uploads**: If you need to collect documents or images, ensure you have the File Storage add-on before creating your form.
