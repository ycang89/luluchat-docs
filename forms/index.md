# Forms

## What is the Forms Module?

The Forms module allows you to build custom, no-code web forms to collect information from your customers. The best part is that data is collected directly from the customer's WhatsApp identity, making it easy to link responses to their profile in Luluchat.

{% hint style="info" %}
**Key Features**

- **No-Code Builder**: Drag-and-drop form creation
- **WhatsApp Integration**: Responses automatically linked to contacts
- **Custom Attributes**: Form data maps directly to contact profiles
- **OTP Verification**: Optional WhatsApp verification for security
{% endhint %}
