# Add an Internal Note

## What is an Internal Note?
An internal-only message in Inbox for teammates. Contacts do not see it.

## When does it trigger?
- When you select `Internal Note` mode in the composer and post a note.

## What happens after it triggers?
The note is visible to teammates in the thread but never sent to the contact.

## Important behavior to know
- Notes are internal-only; contacts do not receive them.
- Tag or mention teammates if supported to draw attention.
