# Automation Logics

## Overview

Logics are non-messaging nodes that control the flow, timing, and data management of your automations. They allow you to create sophisticated customer journeys by branching paths, delaying actions, or performing background updates.

{% hint style="info" %}
**What Logics Do**

- **Control Flow**: Branch paths based on conditions
- **Manage Timing**: Delay actions or wait for events
- **Update Data**: Perform background tasks silently
- **Distribute Work**: Route contacts to team members
{% endhint %}
