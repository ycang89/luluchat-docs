# Starting & Complete Steps

## Overview

Every Message Flow must have a defined beginning and can optionally have explicit end points. These steps define how a customer enters your automation and where the journey successfully concludes.

{% columns %}
{% column %}
**Starting Step**
- Defines entry point
- Sets trigger conditions
- Required for all flows
{% endcolumn %}

{% column %}
**Complete Step**
- Marks flow resolution
- Optional endpoint
- Clear journey conclusion
{% endcolumn %}
{% endcolumns %}
